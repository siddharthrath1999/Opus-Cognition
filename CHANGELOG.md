# Changelog

All notable changes to the Opus-Cognition project will be documented in this master file. 
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-04-16

### Added
- **Opus-Cognition Framework (Opus 4.6)**: 10-Stage cognitive reasoning architecture.
- **Global CLI (`opus`)**: Ephemeral token caching optimization script for local OS.
- **FastMCP Integration**: Native Model Context Protocol server connecting Claude Desktop directly to the reasoning ecosystem.
- **HuggingFace Exporter**: Automated compiler script mapping internal logic tests to dataset arrays for `.jsonl` structural fine-tuning.
- **Automated GitHub Release Pipeline**: Auto-generating Zip and Tar outputs via Github Actions.

### Security / Fixed
- **Docker Isolation Fixes**: Created strict `.dockerignore` shielding to prevent arbitrary RAM bloat by blacklisting `__pycache__` and `.git` mapping.
- **Cross-Platform Protection**: Added formal `.gitattributes` to halt Windows CRLF carriage return text-tearing on adversarial `.js` testing scripts.
- **Universal Local Pathing**: Refactored logic to universally align Python's `__file__` tracing across `tests/`, `packages/`, and `examples/`.
- **CORS Hardening**: Bound independent Playground `index.html` instances to strict Content Security Policies.
- **PyPI Pipeline Constraints**: Protected SDK deployment scripts by formalizing `tools.setuptools` discoveries mapping exactly to the CLI paths.
- **Testing Logic Separation**: Authored a `pytest.ini` guardrail to prevent local runner discovery engines from unintentionally detonating explicit logic bomb payloads.

### Changed
- Converted arbitrary markdown text documents to a fully parameterized DevOps test matrix supported by CI/CD integrations.
- Stamped MIT Legal Headers across the entire programmatic ecosystem.
